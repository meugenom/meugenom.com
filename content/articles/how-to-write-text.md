### Headings

How to use:
```bash
1  # The h1 heading
2  ## The h2 heading
3  ### The h3 heading
4  #### The h4 heading
5  ##### The h5 heading
```

### Quoting text

```bash
    > Quote
    > <cite> - Author </cite>
```

in out:

### Links

You can create an inline link by wrapping link text in brackets, and then wrapping the URL in parentheses:

```bash
	This site was built using [Javascript ES6](https://en.wikipedia.org/wiki/ECMAScript#ES2015)  and it's an example.
```

in out:
This site was built using [Javascript ES6](https://en.wikipedia.org/wiki/ECMAScript#ES2015) and it's an example.

### Ignoring Markdown formatting

You can ignore (or escape) Markdown formatting:

```bash
\* this all text is ### unmarkable\*
this is \*unmarkable\* text
About \*this >Quote\*
```

in out:
\* this all text is ### unmarkable\*
this is \*unmarkable\* text
About \*this >Quote\*
